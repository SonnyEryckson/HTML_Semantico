<header>
	O <header> é utilizado para representar o cabeçalho de um documento ou seção 
declarado no HTML. Nele podemos inserir elementos de <H1> a <h6>, ate 
elementos para representar imagens, paragrafos ou mesmo listas de navegação.

<section>
	O <section> é utilizado para agrupar conteúdo relacionado entre si. (conteúdo independente);

Nota:Cada <section> deve ser identificado, geralmente incluindo um cabeçalho (elemento <h1> 
(en-US)- <h6> (en-US)) como um filho do elemento <section> .

<article>
	O <article> é utilizado para representar para marcar um conteúdo que se constitua em um 
componente autossuficiente em uma página, aplicação ou site e que possa ser distribuido ou 
reusado de forma independente (isolada), como por exemplo via sindicalização
